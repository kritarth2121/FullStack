About Projects

In this project, We have used various properties of HTML and CSS like HTML lists, CSS position, margin, padding etc. This project is divided into three parts.

Firstly, Layout of the whole project is being made by dividing the page with the help of Div tag.

Secondly ,Content is added. We have made horizontal navigation bar with the help of HTML list(Lists are used to group together related pieces of information so they are clearly associated with each other and easy to read.) and then fixed it with position property.The CSS position property is used to set position for an element. it is also used to place an element behind another and also useful for scripted animation effect. We have also made card layouts using to show the technology used.

And in last, Page is made responsive so that users can view this page on any device. For making this page responsive we have used media queries.Media queries were built as an extension to media types commonly found when targeting and including styles. Media queries provide the ability to specify different styles for individual browser and device circumstances, the width of the viewport or device orientation for example. Being able to apply uniquely targeted styles opens up a world of opportunity and leverage to responsive web design.
